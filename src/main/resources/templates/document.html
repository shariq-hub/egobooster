Personalize Ego Booster
HTTP Method GET
URL /api/v1/boosters/{id}/custom
Description [UPDATE] Find a Custom Booster Substituted with a giftee and gifter
Parameter @RequestParam(value = "giftee") @NotNull String giftee
@RequestParam(value = "gifter") String gifter
Sample Response rating (integer)
